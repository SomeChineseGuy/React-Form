### Libraries Installed

- Axios
- React Beautiful DND
- Font Awesome
- @fortawesome/fontawesome-svg-core
- @fortawesome/free-solid-svg-icons
- @fortawesome/react-fontawesome

### Install npm dependencies

```
yarn
```

### Start the app

```
yarn start
```
